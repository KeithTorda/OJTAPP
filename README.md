# 📋 OJT Portal

> Para mapadali ang pag-track ng aking daily OJT records, gumawa ako ng personal na web application na nagbibigay ng structured at visual na paraan ng pag-log ng aking bawat araw sa on-the-job training.

---

## ✨ Features

### 🖥️ Dashboard

- Live training completion ring (progress %)
- Live clock sa TRAINING STATUS bar (12-hour format)
- Metrics: total hours, efficiency, estimated days remaining
- Recent log entries preview
- Export all logs to PDF

### 📝 Daily Log Entry

- Input: Day #, Week #, Date, Duration (hrs), Task Description
- Multi-photo upload with drag & drop support (JPEG, PNG, WebP)
- Fixed SAVE button — always visible kahit nag-scroll
- Live clock + TRAINING STATUS indicator

### 📚 Training Logs

- Complete log history with photo thumbnails
- Click any log to view full details
- Photo lightbox viewer
- Export all logs to PDF

### 🔍 Log Detail & Edit

- View full task description, date, and duration
- Photo gallery with fullscreen lightbox
- Edit any log entry (Day #, Week #, Date, Hours, Description, Photos)
- Add or remove photos per entry
- Export individual log to PDF
- Delete log entry

### 📊 Stats Page

- Visual breakdown of training progress

### 👤 Profile / ID Page

- Trainee information display

### 🔐 Authentication

- Secure login and registration
- Session persistence via localStorage
